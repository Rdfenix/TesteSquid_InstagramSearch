# TesteSquid_InstagramSearch

Passos para utilizar os sistema:

1° no cosole cotendo o diretorio do projeto utilizar:
  npm install
  bower install
 para que o sistema instale os componentes.
 
2º no console do mongo:
  mongod --dbpath "diretorio" caso esteja usando um diretorio externo.
  ou 
  mongopd
 para ativar o banco dados.

3º no diretorio do projeto:
  node server - para rodar o projeto ja configurado.
  
 
 Apos os passos acima o sistema ira requisitar um acesso ao seu instagram para validar o acesso ao sistema, apos isso só realizar as
 pesquisas e salvalas no mongo.
